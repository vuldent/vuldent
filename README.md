<h1>V Team Launcher</h1>
<p>A sleek, feature-rich universal game launcher built with Electron + React.<br/>Add any game. Track everything. Launch in style.</p>
Overview
V Team Launcher started as a mod launcher for Eyes: The Horror Game and evolved into a fully custom universal game launcher. Built from scratch with a focus on clean UI, smooth animations, and features you actually want — session tracking, mood-based launching, play diaries, mod management, and more.
